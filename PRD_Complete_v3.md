## 1. 产品概述

| 属性 | 说明 |
|------|------|
| **产品类型** | 智能医疗助手平台 |

### 3.2 场景过滤层（意图识别前置）

#### 3.2.1 三层过滤流程

```
用户输入
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│  第一层：恶意检测（Security Filter）                      │
│  - SQL注入、XSS攻击、命令注入                            │
│  - 输出：MALICIOUS_INTENT → security-handler             │
└─────────────────────────────────────────────────────────┘
    │ 通过
    ▼
┌─────────────────────────────────────────────────────────┐
│  第二层：场景过滤（Scope Filter）                         │
│  - 非医疗场景识别：天气/股票/新闻/娱乐/技术              │
│  - 输出：OUT_OF_SCOPE → scope-handler                   │
└─────────────────────────────────────────────────────────┘
    │ 通过
    ▼
┌─────────────────────────────────────────────────────────┐
│  第三层：敏感内容过滤（Content Filter）                   │
│  - 政治/色情/暴力/违禁内容                               │
│  - 输出：SENSITIVE_CONTENT → security-handler           │
└─────────────────────────────────────────────────────────┘
    │ 通过
    ▼
┌─────────────────────────────────────────────────────────┐
│  业务意图识别（Business Intent Recognition）              │
│  - 医疗场景意图分类（20种意图类型）                      │
└─────────────────────────────────────────────────────────┘
```

#### 3.2.2 非医疗场景识别规则

| 场景类别 | 关键词/正则模式 | 拒绝话术要点 |
|---------|---------------|-------------|
| 天气查询 | `天气\|气温\|下雨\|下雪\|刮风` | 我专注医疗健康，无法查询天气 |
| 财经股票 | `股票\|基金\|汇率\|理财\|投资` | 我不提供财经资讯服务 |
| 新闻资讯 | `新闻\|热点\|时事\|资讯` | 我不是新闻资讯助手 |
| 娱乐八卦 | `电影\|电视剧\|综艺\|明星\|游戏` | 我不了解娱乐信息 |
| 技术支持 | `电脑\|手机\|软件\|网络\|编程` | 我无法提供技术支持 |
| 生活服务 | `快递\|外卖\|打车\|购物` | 我不提供生活服务 |
| 通用知识 | `为什么.+[^医]\|什么是.+[^医]` | 我专注医疗健康领域 |

### 3.3 意图识别详细规格

#### 3.3.1 症状咨询 (SYMPTOM_INQUIRY)

| 属性 | 规格 |
|------|------|
| **触发条件** | 用户描述身体不适、询问症状原因 |
| **关键词模式 | `我(.+?)(疼\|痛\|难受\|不舒服)`、`(.+?)怎么回事` |
| **必需实体** | body_part（部位）、symptom（症状） |
| **可选实体** | duration（持续时间）、severity（严重程度） |
| **目标Skill** | symptom-analyzer |
| **置信度阈值** | 0.60 |

#### 3.3.2 在线问诊 (ONLINE_CONSULT)

| 属性 | 规格 |
|------|------|
| **触发条件** | 用户想在线咨询医生 |
| **关键词模式** `在线问诊\|图文问诊\|视频问诊\|找医生看` |
| **必需实体** | consult_type（问诊类型） |
| **可选实体** | department（科室）、doctor_type（医生级别） |
| **目标Skill** | online-consult |
| **置信度阈值** | 0.70 |

#### 3.3.3 慢病记录 (CHRONIC_RECORD)

| 属性 | 规格 |
|------|------|
| **触发条件** | 用户记录慢病监测数据 |
| **关键词模式 | `记录(.+?)血压\|血糖\|测量` |
| **必需实体** | disease_type（疾病类型）、value（数值） |
| **可选实体** | measure_time（测量时间）、note（备注） |
| **目标Skill** | chronic-recorder |
| **置信度阈值** | 0.70 |

### 4.2 核心Skill详细规格

#### 4.2.1 症状分析 Skill (symptom-analyzer)

| 属性 | 值 |
|------|-----|
| **功能描述** | 分析用户症状，结合健康档案给出建议 |
| **触发意图** | SYMPTOM_INQUIRY |
| **调用数据服务** | health-records（病史、过敏史） |
| **调用MCP工具** | medical_knowledge_query |

##### 处理流程

```
用户输入："我头痛好几天了"
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│ 1. 实体抽取                                            │
│    - body_part: 头部                                   │
│    - symptom: 疼痛                                     │
│    - duration: 好几天                                  │
└─────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│ 2. 调用健康档案（数据服务层）                           │
│    - 用户有高血压病史？                                 │
│    - 用户有药物过敏史？                                 │
│    - 用户近期类似症状记录？                             │
└─────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│ 3. 调用医学知识库（MCP层）                             │
│    - 头痛的常见原因                                     │
│    - 红旗征识别                                         │
│    - 建议科室                                           │
└─────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│ 4. 综合分析响应                                         │
│    - 结合病史给出针对性建议                             │
│    - 高血压患者头痛需要特殊关注                         │
└─────────────────────────────────────────────────────────┘
```

##### 输入/输出规格

| 输入字段 | 类型 | 必填 | 说明 |
|---------|------|------|------|
| body_part | string | ✅ | 身体部位 |
| symptom | string | ✅ | 症状描述 |
| duration | string | ❌ | 持续时间 |
| severity | string | ❌ | 严重程度 |

| 输出字段 | 类型 | 说明 |
|---------|------|------|
| symptom_analysis | object | 症状分析结果 |
| possible_causes | array | 可能原因 |
| red_flags | array | 红旗征 |
| recommendations | array | 建议（结合病史） |
| suggested_department | string | 建议科室 |
