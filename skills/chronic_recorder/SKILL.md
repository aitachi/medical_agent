# 慢病记录 Skill

当用户需要记录血压、血糖等慢病监测数据时，使用此Skill。

## 触发条件

- 用户要求记录血压/血糖
- 说"量了血压"、"测血糖"
- 提供"血压120/80"等数值
- 询问"血压正常吗"
- 包含"记录"、"测量"、"监测"等关键词

## 支持的慢病类型

### 1. 高血压
**监测指标**:
- 收缩压 (SBP): 正常 < 140 mmHg
- 舒张压 (DBP): 正常 < 90 mmHg
- 心率 (HR): 正常 60-100 bpm

### 2. 糖尿病
**监测指标**:
- 空腹血糖 (FPG): 正常 3.9-6.1 mmol/L
- 餐后2h血糖 (2hPG): 正常 < 7.8 mmol/L
- 糖化血红蛋白 (HbA1c): 正常 < 6%

### 3. 高血脂
**监测指标**:
- 总胆固醇 (TC): 正常 < 5.2 mmol/L
- 甘油三酯 (TG): 正常 < 1.7 mmol/L
- 低密度脂蛋白 (LDL-C): 正常 < 3.4 mmol/L

### 4. 痛风
**监测指标**:
- 血尿酸 (UA): 正常 150-420 μmol/L

## 处理流程

### 1. 识别慢病类型

从用户输入中识别：
- 关键词："血压"→ 高血压
- 关键词："血糖"→ 糖尿病
- 关键词："尿酸"→ 痛风

### 2. 解析测量数据

支持自然语言格式：
- "血压 120/80" → {systolic: 120, diastolic: 80}
- "空腹血糖 6.5" → {fasting_glucose: 6.5}
- "血压140/90心率85" → {systolic: 140, diastolic: 90, heart_rate: 85}

### 3. 对比参考范围

判断每个指标状态：
- 正常: 在参考范围内
- 偏高/偏低: 轻度超出范围
- 高/低: 明显超出范围
- 危急: 严重超出范围

### 4. 趋势分析

对比最近7天数据：
- 上升 📈: 数值呈上升趋势
- 下降 📉: 数值呈下降趋势
- 稳定 ➡️: 数值保持稳定

### 5. 异常告警

根据指标严重程度发出告警：
- 正常: 无告警
- 偏高: ⚠️ 指标偏高，请注意监测
- 偏高: ⚠️ 指标偏高，建议及时就医
- 危急: 🚨 指标异常，请立即就医

### 6. 健康建议

根据指标状态提供针对性建议。

## 示例对话

**用户**: 刚量了血压，140/90

**处理过程**:
1. 识别类型: 高血压
2. 解析数据: 收缩压140, 舒张压90
3. 分析状态: 收缩压偏高, 舒张压正常偏高
4. 生成建议

**响应**: 异常记录响应

**用户**: 我的空腹血糖是7.0

**处理过程**:
1. 识别类型: 糖尿病
2. 解析数据: 空腹血糖7.0
3. 分析状态: 偏高
4. 生成建议

**响应**: 糖尿病记录响应

## 注意事项

- 不给出确定性诊断
- 危急值时强调立即就医
- 保持客观、专业
- 建议定期监测
- 必要时建议就医
- 保持记录的连续性
