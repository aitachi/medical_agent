# 科室推荐 Skill

当用户询问应该挂什么科室时，使用此Skill。

## 触发条件

- 用户问"挂什么科"、"去哪个科室"
- 询问"XX症状看什么科"
- 询问"XX病挂哪个科"
- 想要了解医院科室设置

## 处理流程

### 1. 分析用户需求

判断用户意图：
- 根据症状推荐科室
- 查看科室列表
- 查看科室详情

### 2. 调用MCP工具

**根据症状推荐**:
```
hospital_department_query(
  query_type: "by_symptom",
  symptom: <用户描述的症状>
)
```

**查看科室列表**:
```
hospital_department_query(
  query_type: "list"
)
```

**查看科室详情**:
```
hospital_department_query(
  query_type: "detail",
  department: <科室名称>
)
```

### 3. 生成响应

**症状推荐响应格式**:
```markdown
根据您描述的「{症状}」，建议挂以下科室：

🏥 **{科室1}**
- 适用情况: {描述}

🏥 **{科室2}**
- 适用情况: {描述}

💡 **挂号提示**:
- {科室}位于{位置}
- {其他提示}
```

**科室列表响应格式**:
```markdown
本院开设以下科室：

| 科室 | 简介 |
|------|------|
| {科室1} | {简介} |
| {科室2} | {简介} |
| ...
```

## 示例对话

**用户**: 头疼应该挂什么科？

**处理**:
1. 识别意图: 科室推荐
2. 提取症状: 头疼
3. 调用MCP: hospital_department_query(by_symptom=头疼)
4. 生成响应

**响应**:
根据「头疼」症状，建议挂以下科室：

🏥 **神经内科**
- 适用情况: 头痛、头晕、失眠等神经系统症状

🏥 **普通内科**
- 适用情况: 一般性头痛、全身不适

⚠️ **注意**: 如果头痛剧烈或突发，请直接到急诊科就诊。

## 科室映射规则

| 症状关键词 | 推荐科室 |
|-----------|---------|
| 头痛、头晕 | 神经内科 |
| 胸痛、心悸 | 心血管内科 |
| 咳嗽、气促 | 呼吸内科 |
| 腹痛、恶心 | 消化内科 |
| 皮疹、瘙痒 | 皮肤科 |
| 发热 | 发热门诊 |

## 注意事项

- 多个症状可能对应多个科室，列出所有可能选项
- 紧急症状建议急诊科
- 儿童患者建议儿科
- 妇科相关问题建议妇产科
