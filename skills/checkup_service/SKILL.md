# 体检服务 Skill

当用户询问体检套餐、预约体检、体检注意事项时，使用此Skill。

## 触发条件

- 询问体检套餐
- 要求预约体检
- 询问体检注意事项
- 询问"体检做什么"
- 包含"体检"、"检查"、"筛查"等关键词

## 支持的体检类型

### 1. 基础体检
- 青年基础体检（18-35岁）
- 中年基础体检（36-50岁）
- 价格：¥299-499

### 2. 全面体检
- 全面体检套餐
- 价格：¥1299
- 适合：40岁以上、有家族病史

### 3. 高端体检
- 高端体检套餐
- 价格：¥2999
- 适合：高管、高净值人群

### 4. 针对性体检
- 心血管专项
- 女性专项
- 男性专项
- 价格：¥599-899

## 处理流程

### 推荐套餐

1. 收集用户信息（年龄、性别、健康关注）
2. 根据条件筛选套餐
3. 展示推荐套餐
4. 提供预约信息

### 预约提示

提供体检前注意事项：
- 饮食要求
- 空腹要求
- 特殊情况处理

## 响应格式

### 套餐推荐响应

```markdown
## 🏥 体检套餐推荐

### 👤 根据您的情况

- **年龄**: 35岁
- **性别**: 女
- **健康关注**: 妇科, 乳腺

### 推荐1: 女性专项体检

- **套餐类型**: targeted
- **价格**: ¥699
- **所需时间**: 半天
- **适用人群**: 25岁以上女性, 已婚女性

**检查项目**:
- 基础体检项目
- 妇科检查
- 白带常规
- TCT（液基薄层细胞检测）
- HPV检测
- 乳腺彩超
- 妇科彩超（子宫、附件）
- 骨密度检测
- 女性激素评估
- 乳腺、宫颈健康指导

### 📅 预约提示

- 体检前3天清淡饮食，避免饮酒
- 体检前1晚20点后禁食
- 体检当日早晨空腹（可少量饮水）
- 女性避开月经期
- 佩戴者可正常服药（少量水）
```

## 注意事项

- 女性体检注意避开月经期
- 孕妇避免放射性检查
- 空腹项目提醒
- 佩戴设备者注意事项
- 检查前停药提醒
