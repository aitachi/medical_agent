# 健康教育 Skill

提供健康知识科普和疾病预防建议。

## 触发条件

- 询问如何预防某种疾病
- 询问健康生活方式
- 询问饮食禁忌
- 询问运动建议
- 询问健康保健知识

## 知识库

### 高血压预防

**预防措施**:
1. **饮食控制**
   - 低盐饮食（每日盐摄入<6g）
   - 低脂饮食
   - 多吃蔬菜水果

2. **生活方式**
   - 规律运动（每周3-5次，每次30分钟）
   - 控制体重（BMI 18.5-24）
   - 戒烟限酒

3. **定期监测**
   - 定期测量血压
   - 40岁以上每年检查

**饮食禁忌**:
- 高盐食物: 咸菜、腌制品、方便面
- 高脂食物: 动物内脏、油炸食品
- 刺激性饮料: 浓茶、咖啡

### 糖尿病预防

**预防措施**:
1. **饮食管理**
   - 控制碳水化合物摄入
   - 低糖饮食
   - 少量多餐

2. **运动锻炼**
   - 每周至少150分钟中等强度运动
   - 避免久坐

3. **体重控制**
   - 维持理想体重
   - 腰围控制（男性<90cm，女性<85cm）

**饮食禁忌**:
- 高糖食物: 糖果、蛋糕、甜饮料
- 精制碳水: 白米饭、白面包
- 高淀粉蔬菜: 土豆、红薯

### 心血管疾病预防

**预防措施**:
1. **健康饮食**
   - 地中海饮食模式
   - 增加膳食纤维
   - 适量优质蛋白

2. **规律运动**
   - 有氧运动为主
   - 循序渐进

3. **危险因素控制**
   - 控制三高（高血压、高血糖、高血脂）
   - 戒烟
   - 管理压力

### 呼吸系统健康

**预防措施**:
1. **避免刺激**
   - 戒烟
   - 避免二手烟
   - 空气污染时戴口罩

2. **增强免疫**
   - 规律作息
   - 均衡营养
   - 适度运动

3. **定期检查**
   - 肺功能检查
   - 低剂量CT（高危人群）

### 消化系统健康

**预防措施**:
1. **饮食卫生**
   - 规律饮食
   - 细嚼慢咽
   - 避免过烫食物

2. **生活习惯**
   - 餐后适当活动
   - 避免暴饮暴食
   - 限制饮酒

3. **定期筛查**
   - 胃镜检查（40岁以上）
   - 大便潜血试验

### 睡眠健康

**睡眠建议**:
1. **睡眠时长**
   - 成人: 7-9小时
   - 老年人: 7-8小时
   - 儿童: 9-11小时

2. **睡眠习惯**
   - 固定作息时间
   - 睡前避免电子产品
   - 营造良好睡眠环境

3. **改善睡眠**
   - 规律运动
   - 下午后避免咖啡因
   - 睡前放松活动

### 运动建议

**运动类型**:
1. **有氧运动**
   - 快走、慢跑、游泳、骑自行车
   - 每周150分钟中等强度或75分钟高强度

2. **力量训练**
   - 每周2-3次
   - 主要肌群训练

3. **柔韧性训练**
   - 每周2-3次拉伸
   - 瑜伽、太极

**运动注意事项**:
- 运动前热身
- 循序渐进
- 身体不适时暂停
- 运动后放松

## 通用健康建议

### 日常习惯
1. **三餐规律** - 早餐要吃好，晚餐要吃少
2. **充足饮水** - 每日1.5-2升
3. **规律作息** - 早睡早起
4. **适度运动** - 每日30分钟
5. **心情愉快** - 管理压力

### 饮食原则
1. **食物多样** - 每日12种以上
2. **谷类为主** - 全谷物更好
3. **多吃蔬果** - 每日500克
4. **适量蛋白** - 鱼禽蛋奶豆
5. **少油少盐** - 油25-30g，盐<6g

### 运动原则
1. **循序渐进** - 从少量开始
2. **持之以恒** - 形成习惯
3. **量力而行** - 不勉强
4. **全面发展** - 有氧+力量+柔韧
