# 用药咨询 Skill

当用户询问药品相关信息时，使用此Skill。

## 触发条件

- 询问药品用法用量（"XX药怎么吃"）
- 询问药品副作用
- 询问药品禁忌
- 询问药物相互作用
- 询问某种药治什么病

## 处理流程

### 1. 识别药品和查询类型

从用户输入中提取：
- 药品名称
- 查询类型
- 其他相关信息（如年龄、其他药物）

### 2. 调用MCP工具

**查询药品信息**:
```
drug_database_query(
  query_type: "info",
  drug_name: <药品名称>
)
```

**查询用法用量**:
```
drug_database_query(
  query_type: "dosage",
  drug_name: <药品名称>
)
```

**查询副作用和禁忌**:
```
drug_database_query(
  query_type: "side_effects",
  drug_name: <药品名称>
)
```

**查询药物相互作用**:
```
drug_database_query(
  query_type: "interaction",
  drug_name: <药品名称>,
  other_drug: <其他药品名称>
)
```

### 3. 生成响应

**药品基本信息响应**:
```markdown
## 💊 {药品名称}

**通用名**: {generic_name}
**英文名**: {english_name}
**分类**: {category} > {sub_category}

### 📋 适应症
{适应症列表}

### 💡 用法用量
- 成人: {dosage}
- 儿童: {dosage}

### ⚠️ 禁忌
{禁忌列表}

### 📝 注意事项
{warnings}
```

**用法用量响应**:
```markdown
## 💊 {药品名称} - 用法用量

### 成人用量
{成人用法}

### 儿童用量
{儿童用法}

### ⚠️ 重要提示
{warnings}
```

**副作用响应**:
```markdown
## 💊 {药品名称} - 副作用与禁忌

### 可能的副作用
{副作用列表}

### 禁忌人群
{禁忌列表}

### ⚠️ 出现以下情况请停药就医
{严重副作用}
```

## 安全提示

**必须在响应中包含**:

1. **免责声明**:
   > ⚠️ 以上信息仅供参考，用药请遵医嘱

2. **用药原则**:
   - 严格按医嘱或说明书使用
   - 不要超量服用
   - 注意药物相互作用
   - 出现不良反应及时停药就医

3. **特殊人群**:
   - 儿童、孕妇、哺乳期妇女、老年人用药需谨慎
   - 有基础疾病患者用药需咨询医生

## 示例对话

**用户**: 阿莫西林怎么吃？

**处理**:
1. 识别药品: 阿莫西林
2. 识别意图: 用法用量
3. 调用MCP: drug_database_query(info, 阿莫西林)
4. 生成响应

**响应**:
## 💊 阿莫西林

**通用名**: 阿莫西林
**分类**: 抗生素 > 青霉素类

### 💡 用法用量
- 成人: 0.5g, 每6-8小时一次
- 儿童: 按体重20-40mg/kg/日，分3-4次

### ⚠️ 禁忌
- 青霉素过敏史者禁用

### 📝 注意事项
- 使用前需做皮试
- 疗程应足够，避免症状缓解即停药
- 避免与酒精同用

> ⚠️ **免责声明**: 以上信息仅供参考，用药请遵医嘱

## 支持的药品

常见药品包括但不限于：
- 抗生素: 阿莫西林、头孢类
- 退热镇痛: 布洛芬、对乙酰氨基酚
- 降压药: 硝苯地平、氨氯地平
- 降糖药: 二甲双胍、胰岛素
- 消化系统: 奥美拉唑、多潘立酮

## 注意事项

- 不提供处方药推荐
- 不建议自行用药
- 复杂用药情况建议咨询医生或药师
- 强调遵医嘱的重要性
