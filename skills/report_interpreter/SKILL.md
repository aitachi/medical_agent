# 报告解读 Skill

当用户提供检查报告内容，需要解读指标含义和异常情况时，使用此Skill。

## 触发条件

- 用户上传或描述检查报告内容
- 询问某项指标的含义
- 询问指标是否正常
- 询问异常指标的临床意义
- 包含"报告"、"检查"、"化验"、"结果"、"指标"、"偏高"、"偏低"等关键词

## 支持的报告类型

### 1. 血常规
- 白细胞 (WBC)
- 红细胞 (RBC)
- 血红蛋白 (HGB)
- 红细胞压积 (HCT)
- 血小板 (PLT)
- 中性粒细胞百分比 (NEUT%)
- 淋巴细胞百分比 (LYMPH%)

### 2. 尿常规
- 尿蛋白 (U_PRO)
- 尿糖 (U_GLU)
- 尿潜血 (U_BLD)
- 尿白细胞 (U_LEU)
- 尿酮体 (U_KET)
- 尿pH值

### 3. 肝功能
- 谷丙转氨酶 (ALT)
- 谷草转氨酶 (AST)
- 总蛋白 (TP)
- 白蛋白 (ALB)
- 球蛋白 (GLB)
- 总胆红素 (TBIL)
- 直接胆红素 (DBIL)

### 4. 肾功能
- 肌酐 (CRE)
- 尿素 (UREA)
- 尿酸 (UA)
- 肾小球滤过率 (eGFR)

### 5. 血糖
- 空腹血糖 (GLU)
- 糖化血红蛋白 (HbA1c)
- 餐后2小时血糖 (2hPG)

### 6. 血脂
- 总胆固醇 (TC)
- 甘油三酯 (TG)
- 高密度脂蛋白 (HDL-C)
- 低密度脂蛋白 (LDL-C)

### 7. 甲状腺功能
- 促甲状腺激素 (TSH)
- 游离T3 (FT3)
- 游离T4 (FT4)
- 总T3 (T3)
- 总T4 (T4)

## 处理流程

### 1. 识别报告类型

从用户输入中识别报告类型：
- 关键词匹配（如"血常规"、"肝功能"）
- 指标名称匹配（如WBC、ALT、GLU）
- 用户指定类型

### 2. 解析指标数据

从报告文本中提取：
- 指标名称
- 检测值
- 单位
- 参考范围

### 3. 对比参考范围

判断每个指标是否异常：
- 高于参考上限 → 标记为升高 (↑)
- 低于参考下限 → 标记为降低 (↓)
- 阳性/阴性 → 标记状态

### 4. 生成解读内容

#### 报告摘要
- 报告类型
- 异常指标数量
- 异常指标列表

#### 异常指标分析（如有）
每项异常指标包含：
- 指标名称和异常标识
- 检测值和参考范围
- 临床意义说明
- 针对性建议

#### 综合评估
- 整体健康状况评价
- 异常指标的综合分析

#### 建议
- 针对异常指标的生活建议
- 饮食建议
- 复查建议

#### 随访建议
- 是否需要随访
- 复查时间建议
- 是否需要专科就诊

### 5. 添加免责声明

必须在响应末尾添加：
> ⚠️ **免责声明**: 以上解读仅供参考，不能替代专业医生的诊断。如有疑问，请及时咨询相关专科医生。

## 响应格式

### 正常报告示例

```markdown
# 📋 检查报告解读

**报告类型**: blood_routine

**报告摘要**: 血常规报告未见明显异常，各项指标均在正常范围内。

## ✅ 各项指标正常

您的各项指标均在正常范围内，请继续保持健康的生活方式。

## 📊 综合评估

您的检查结果基本正常，建议继续保持健康的生活方式，定期体检。

---

> ⚠️ **免责声明**: 以上解读仅供参考，不能替代专业医生的诊断。如有疑问，请及时咨询相关专科医生。
```

### 异常报告示例

```markdown
# 📋 检查报告解读

**报告类型**: blood_routine

**报告摘要**: 血常规报告发现2项指标异常：白细胞, 中性粒细胞%

## ⚠️ 异常指标

### 白细胞 ↑

- **检测值**: 12.5 10^9/L
- **参考范围**: 4.0-10.0
- **临床意义**: 白细胞升高可能提示细菌感染、炎症或应激反应
- **建议**: 多喝水，注意休息，避免过度劳累，如有发热及时就医

### 中性粒细胞% ↑

- **检测值**: 75%
- **参考范围**: 50-70
- **临床意义**: 中性粒细胞比例升高常见于细菌感染
- **建议**: 结合临床症状评估，必要时查感染指标

## 📊 综合评估

您的检查结果存在以下情况：
- 白细胞升高：白细胞升高可能提示细菌感染、炎症或应激反应
- 中性粒细胞%升高：中性粒细胞比例升高常见于细菌感染

## 💡 建议

- 多喝水，注意休息
- 避免过度劳累
- 如有发热及时就医
- 建议1-2周后复查异常指标
- 注意休息，避免过度劳累
- 如有疑问，请咨询相关专科医生

## 📅 随访建议

建议1-2周后复查相关指标，必要时专科就诊

---

> ⚠️ **免责声明**: 以上解读仅供参考，不能替代专业医生的诊断。如有疑问，请及时咨询相关专科医生。
```

## 示例对话

**用户**: 帮我看看这个血常规报告
```
WBC: 12.5 10^9/L
RBC: 4.5 10^12/L
HGB: 135 g/L
```

**处理过程**:
1. 识别报告类型: 血常规
2. 解析指标: WBC=12.5 (偏高), RBC=4.5 (正常), HGB=135 (正常)
3. 生成解读

**响应**: (如上方异常报告示例)

## 注意事项

- 不要给出确定性诊断，只能给出参考性解读
- 遇到危急值指标时，强调及时就医
- 保持语气专业、客观
- 必须包含免责声明
- 不要推荐具体药物
- 多项异常时建议专科就诊
